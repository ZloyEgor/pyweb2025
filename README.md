# Курс "Веб-технологии на Python" 2025

Данный репозиторий содержит выполненные задания по курсу веб-технологий с использованием языка программирования Python.

## Структура репозитория

Каждое задание размещено в отдельной директории с соответствующей нумерацией и содержит полную документацию по реализации, включая техническое описание, инструкции по развертыванию и анализ используемых технологий.

## Выполненные задания

### [Задание 2.1 - Создание и развертывание статического сайта](task2.1/)

**Описание:** Разработка и автоматизированное развертывание статического веб-сайта с использованием генератора MkDocs и системы непрерывной интеграции GitHub Actions.

**Технологический стек:**
- Python 3.x
- MkDocs с темой Material Design
- GitHub Actions для CI/CD
- GitHub Pages для хостинга

**Результат:** Функционирующий статический сайт с автоматическим обновлением при внесении изменений в репозиторий.

**Демонстрация:** https://zloyegor.github.io/pyweb2025/
